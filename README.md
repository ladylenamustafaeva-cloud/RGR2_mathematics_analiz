# RGR1_mathematics_analiz
# Численное интегрирование: методы Ньютона-Котеса

Расчетно-графическая работа №1 по численным методам.

## Описание
Реализация четырёх методов численного интегрирования:
- Метод прямоугольников (средних точек)
- Метод трапеций
- Метод Симпсона (парабол)
- Метод «трёх восьмых»

## Структура проекта
| Файл | Описание |
|------|----------|
| `methods.py` | Реализация методов интегрирования |
| `plots.py` | Построение графиков зависимости погрешности и трудоёмкости |
| `main.py` | Точка входа: запуск численного эксперимента |
