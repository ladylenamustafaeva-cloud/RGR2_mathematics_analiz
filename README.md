# RGR1_mathematics_analiz
# Численное интегрирование: методы Ньютона-Котеса

## 📁 Структура проекта

### `methods.py` — ядро вычислений
Реализует 4 метода численного интегрирования + адаптивный подбор шага.

**Функции:**
- `rectangles(f, a, b, n)` → float
- `trapezoid(f, a, b, n)` → float  
- `simpson(f, a, b, n)` → float (автоматически делает n чётным)
- `three_eighths(f, a, b, n)` → float (автоматически делает n кратным 3)
- `adaptive_integration(method, f, a, b, eps)` → (result, n)

### `plots.py` — визуализация
- `plot_error_vs_epsilon(...)` — график погрешности от требуемой точности
- `plot_n_vs_epsilon(...)` — график трудоёмкости
- `plot_convergence(...)` — график сходимости методов

### `main.py` — запуск
Запускает полный эксперимент: вычисления + графики + таблицы.
