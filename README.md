# RGR1_mathematics_analiz
# Численное интегрирование: методы Ньютона-Котеса

Расчётно-графическая работа №1 по численным методам.

## Описание

Реализация четырёх методов численного интегрирования для вычисления определённых интегралов с заданной точностью:
- Метод прямоугольников
- Метод трапеций
- Метод Симпсона
- Метод «трёх восьмых»

## Цели исследования

1. Исследовать влияние заданной точности ε на объём вычислений
2. Исследовать влияние гладкости функции на точность вычислений
3. Сравнить теоретическую и фактическую погрешности методов

## Модули программы

### `methods.py`
| Функция | Входные параметры | Выходные параметры | Описание |
|---------|------------------|------------------|----------|
| `rectangles(f, a, b, n)` | `f`: функция, `a,b`: пределы, `n`: число разбиений | `float`: значение интеграла | Метод прямоугольников (средние точки) |
| `trapezoid(f, a, b, n)` | аналогично | `float`: значение интеграла | Метод трапеций |
| `simpson(f, a, b, n)` | аналогично (n автоматически делается чётным) | `float`: значение интеграла | Метод Симпсона |
| `three_eighths(f, a, b, n)` | аналогично (n автоматически делается кратным 3) | `float`: значение интеграла | Метод 3/8 |
| `adaptive_integration(method, f, a, b, eps)` | `method`: функция метода, `eps`: требуемая точность | `(result, n)`: значение интеграла и число разбиений | Адаптивный подбор шага |

### `plots.py`
| Функция | Назначение |
|---------|-----------|
| `plot_error_vs_epsilon(...)` | График зависимости фактической погрешности от требуемой точности ε |
| `plot_n_vs_epsilon(...)` | График зависимости числа разбиений N от требуемой точности ε |

### `convergence.py`
| Функция | Назначение |
|---------|-----------|
| Основной блок | Построение графика сходимости: погрешность от числа разбиений N |

| Функция | Отрезок | Точное значение | Свойства |
|---------|---------|----------------|----------|
| `f₁(x) = eˣ` | [0, 1] | `e − 1 ≈ 1.71828` | Гладкая, бесконечно дифференцируема |

## Краткие выводы

- Методы Симпсона и 3/8 (порядок точности `O(h⁴)`) достигают высокой точности при малом числе разбиений
- Методы прямоугольников и трапеций (порядок `O(h²)`) требуют значительно больше вычислений для той же точности
- На негладких функциях методы высокого порядка теряют преимущество: фактическая погрешность не соответствует теоретическим оценкам
